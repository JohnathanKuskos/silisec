h1. This is the data for silisec page

It is automatically transformed by "Jekyll":http://github.com/mojombo/jekyll into a static site whenever I push this repository to GitHub.

h1. How to Use/Add a Silisec Date

h2. Name of the File

Add a file to the posts folder with the following format:

YYYY-MM-DD-MONTH.textile (MONTH is the full name of the month that the silisec will be happening, i.e. 2011-01-01-August.textile)

h2. Contents

The contents should contain the following. Everything that needs to be modified is in bold:

---
layout: post
title: *June 2011* Silisec
---

h1. *June 2nd, 2011*

*The June silisec meetup is around 7pm on June 2nd at the Faultline Brewing Company in Sunnyvale.*
